# 🛡️ Observer Bot

Анализ безопасности транзакций Solana прямо в Telegram.

[Открыть бота](https://t.me/observerprojectbot)

---

## 📌 О проекте

**Observer Bot** — это Telegram-бот для анализа транзакций в сети Solana.  
Он помогает выявлять потенциально опасные действия и скрытые риски.

Бот определяет:
- передачу прав (Approve, SetAuthority)
- подозрительные переводы
- возможный фишинг
- рискованные действия в логах транзакции

---

## 🚀 Возможности

- Анализ транзакций по Signature  
- Быстрый ответ в Telegram  
- Оценка риска: LOW / MEDIUM / HIGH  
- Простая система скоринга  
- Работа через Solana RPC  

---

## 🤖 Как пользоваться

1. Откройте бота: https://t.me/observerprojectbot  
2. Введите команду: 
/start

3. Отправьте Signature транзакции  
4. Получите отчет  

---

## 📊 Пример ответа


🛡 Отчет безопасности

Риск: HIGH 🔴
Баллы: 5


### Уровни риска

- LOW 🟢 - угроз не обнаружено  
- MEDIUM 🟡 - есть подозрительные действия  
- HIGH 🔴 - высокий риск

## ⚙️ Установка


git clone https://github.com/mrnftsol2011/observer.git

cd observer
pip install -r requirements.txt


---

## 🔐 Настройка

Создайте файл `.env`:


BOT_TOKEN=your_token
RPC_URL=https://api.mainnet-beta.solana.com

## ▶️ Запуск


python src/main.py
