# 🛡️ Observer Bot (@observerprojectbot)

**Observer Bot** — это ваш персональный аудит безопасности в телеграм. Бот анализирует транзакции в сети Solana и выявляет скрытые угрозы, такие как попытки кражи прав доступа или неограниченные разрешения на трату ваших токенов.

---

## 🤖 Как пользоваться ботом

1. Найдите в Telegram бота: **[@observerprojectbot](https://t.me/observerprojectbot)**
2. Нажмите кнопку **/start**.
3. Скопируйте **Signature** (ID) любой транзакции из вашего кошелька (Phantom, Solflare) или с сайта [Solscan](https://solscan.io).
4. Отправьте Signature боту сообщением.

### Возможный ответ:
* **LOW 🟢** — Обычная транзакция, угроз не обнаружено.
* **MEDIUM 🟡** — Есть подозрительные действия (например, передача прав или аппрув).
* **HIGH 🔴** — Критическая опасность! Подозрение на фишинг или кражу активов.
